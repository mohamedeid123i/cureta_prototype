A comprehensive interactive mobile prototype for a healthcare application built with HTML, CSS, and JavaScript. Includes authentication, dashboard, AI chatbot, prescription scanning, and health records management.

## Project Structure

- **`prototype/`** - Main prototype entry point with navigation flows
- **`stitch_create_account/`** - All screen components (60+ screens)
  - Onboarding & Authentication (Phase 1)
  - Dashboard & Navigation (Phase 2)
  - Health Management Features

## Features

✅ Responsive mobile UI (393×852px - iPhone 15)
✅ Interactive chatbot with AI responses
✅ Multi-screen navigation system
✅ Side drawer navigation
✅ Bottom navigation bar
✅ Toast notifications
✅ Cross-platform compatibility

## How to Run

1. Start a local web server:
```bash
cd App_Prototype
python -m http.server 8080
```

2. Open in browser:
```
http://localhost:8080/prototype/
```

3. Navigate through screens using buttons and menu items

## Tech Stack

- CSS3 (Tailwind CSS)
- Vanilla JavaScript
- Figma-exported components
